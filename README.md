# SERVIVEMKA 2618 CA - Sitio web base

Este proyecto es una web estática lista para subir a GitHub y publicar en Vercel.

## Archivos

- `index.html`: estructura principal de la página
- `styles.css`: estilos completos y paleta de colores
- `script.js`: menú móvil, FAQ interactiva y formulario de demostración
- `public/logo-servivemka.jpg`: logo actual de la empresa

## Qué debes cambiar antes de publicar

Busca y reemplaza estos datos:

- `0412 000 0000`
- `tucorreo@servivemka.com`
- `Venezuela` si quieres una ciudad específica
- enlace de WhatsApp dentro de `index.html`

## Cómo subir a GitHub

1. Crea un repositorio nuevo en GitHub.
2. Sube todos estos archivos.
3. Verifica que `index.html` esté en la raíz del repositorio.

## Cómo publicarlo en Vercel

1. Inicia sesión en Vercel.
2. Importa el repositorio desde GitHub.
3. Como es un sitio estático, Vercel lo detectará automáticamente.
4. Pulsa deploy.

## Personalizaciones recomendadas después

- conectar el formulario a correo o WhatsApp real
- agregar testimonios
- agregar galería de trabajos
- incluir mapa de ubicación
- añadir favicon y redes sociales
